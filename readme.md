# MPK Ticket Machine

## Instalation

- Windows
```
git clone https://github.com/kacxsku/MPKTicketMachine.git
cd MPKTicketMachine
py -3 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

- Linux
```
git clone https://github.com/kacxsku/MPKTicketMachine.git
cd MPKTicketMachine
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt
```
