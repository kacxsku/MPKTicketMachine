# MPK Ticket Machine

## Instalation

- Windows
```
git clone https://github.com/szymonszoldra/battleship.git
cd battleship
py -3 -m venv venv
venv\Scripts\activate
pip install -r requirements.tx
```

- Linux
```
git clone https://github.com/szymonszoldra/battleship.git
cd battleship
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt
```
